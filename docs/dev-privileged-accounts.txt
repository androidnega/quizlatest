----------------------------------------------------------------------
ROLES (table: roles) — created by InitialSetupSeeder
----------------------------------------------------------------------
- Admin          slug: admin
- Coordinator    slug: coordinator

----------------------------------------------------------------------
PERMISSIONS (table: permissions) — created by InitialSetupSeeder
----------------------------------------------------------------------
- Examiner   slug: examiner   (coordinator role may hold this via permission_role)
- Student    slug: student     (linked to student role)

permission_role links:
- coordinator  -> examiner permission
- student      -> student permission

(Admins use users.role = "admin" in code; see policies for exact gates.)
